Rajagiri Event Management System

Rajagiri Event Management System is a platform designed to help students and teachers view and manage upcoming events and activity points efficiently. It integrates advanced filtering capabilities to enhance user experience and simplify the event management process.

System Overview

The Rajagiri Event Management System offers a comprehensive solution for managing events within the institution. It provides features for both students and teachers, facilitating easy access to upcoming events, activity points, and certificate submissions. The system aims to streamline event organization and activity tracking.

Key Features

  • Login for Students and Teachers: Separate login portals for students and teachers to access personalized information.
  • Display of Events: A view of all upcoming events, making it easier to stay informed.
  • Event Management: Allows users to add and delete events with ease.
  • Advanced Filtering: Enhanced filtering options using AI for a more personalized experience.
  • Activity Points Management: Submit certificates for activity points and track progress.
  • Review Feature: Users can review event details and view itineraries.
